# Devil May Cry – Página informativa

Este proyecto es una página web informativa inspirada en la saga de videojuegos Devil May Cry. Fue desarrollada utilizando el framework Bootstrap 5 con el objetivo de crear una página responsiva y bien estructurada.

El sitio forma parte de un ejercicio académico enfocado en el uso de frameworks CSS y la publicación de sitios web estáticos.

---

## Descripción

La página presenta información general sobre la saga Devil May Cry, incluyendo:

- Una introducción a la franquicia
- Un resumen de la historia
- Descripción de los personajes principales
- Información general sobre los juegos
- Una explicación del estilo de combate característico

Para el diseño se emplearon componentes de Bootstrap como la barra de navegación, el sistema de rejillas, tarjetas y alertas.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5 (incluido mediante CDN con jsDelivr)
- GitHub Pages para el despliegue del sitio
